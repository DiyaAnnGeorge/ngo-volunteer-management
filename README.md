# 🌿 NGO Connect — Volunteer Management System

A React-based web application for managing NGO volunteer activities at VIT Vellore.

## 🌟 Features

### 👤 Volunteer
- Sign up / Log in
- Browse NGO clubs and their events
- Join events and track registered activities
- View volunteer hours and achievements dashboard

### 🔐 Admin
- Secure admin portal
- Create and delete events across clubs
- Assign events to specific NGO clubs

### 📋 Coordinator
- Mark volunteer attendance at events
- Approve volunteer hours after attendance
- View all registrations per event

## 🛠️ Tech Stack
- **Frontend:** React.js, React Router
- **Styling:** CSS
- **Icons:** Lucide React
- **State Management:** React Hooks (useState, useEffect)

## ⚙️ Setup Instructions

1. Clone the repository
   git clone https://github.com/DiyaAnnGeorge/ngo-volunteer-management.git

2. Install dependencies
   npm install

3. Start the development server
   npm start
